# Code of Conduct

## Our commitment

We want this project to be welcoming and respectful for everyone.

## Expected behavior
- Be constructive and kind.
- Assume good intent.
- Give specific, actionable feedback.
- Focus on improving code and user outcomes.

## Unacceptable behavior
- Harassment or personal attacks.
- Discriminatory or hateful language.
- Publishing private information without consent.
- Repeated disruptive behavior.

## Enforcement

Maintainers may remove comments, close threads, or block contributors who violate this policy.
