O cenário típico (Obsidian)

Você tem notas emreduçãocom LaTeX (inline e bloco). Ao mudar para o Word, o problema comum é que as equações permanecem como texto (você vê o$) ou não são editáveis.

Rota recomendada

  1. Exporte sua notacomo.md(Remarcação).
  2. Converter para .docx(por exemplo, com Pandoc) para manter a estrutura básica.
  3. Delimitadores segurose converta o documento para OMML comEquações para Palavra.

Lista de verificação rápida se algo der errado

  • Verifique se não há blocos de equações de encapsulamento de código.
  • Evite macros LaTeX personalizadas.
  • Se Pandoc sair$...$visível, aplique esta solução de problemas:Pandoc: equações não editáveis.

Se você também quiser entender por que o Word precisa de OMML para equações editáveis, leia:O que é OML.
