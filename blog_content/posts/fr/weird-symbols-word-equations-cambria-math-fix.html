Symptôme

Les équations semblent correctes sur votre ordinateur, mais sur un autre appareil, elles s'affichentboîtes carrées, des symboles remplacés ou des caractères inattendus.

Cause la plus courante : incompatibilité des polices mathématiques

Les équations de mots reposent généralement surMathématiques Cambria(ou polices mathématiques équivalentes). Si la police est manquante ou remplacée par un style de modèle agressif, les symboles peuvent se briser.

Correctifs par ordre de priorité

  1. Sélectionnez l'équation et confirmez qu'une police compatible avec les mathématiques est appliquée (Cambria Math).
  2. Évitez de forcer une police de corps globale dans les exécutions d’équations.
  3. Rouvrez le fichier dans le bureau Word si le problème apparaît dans les applications d'aperçu.
  4. Préférez la conversion LaTeX vers OMML au lieu du collage de texte riche mixte à partir des discussions AI.

Si la source est ChatGPT

Si vous voyez également des points d'interrogation ou des symboles impairs après un copier/coller, vérifiez :points d'interrogation lors du collage d'équations ChatGPT.

Liste de contrôle de livraison stable

  • Convertissez votre fichier source en équations OMML.
  • Révisez le .docx final dans la même version Word utilisée pour la notation/révision.
  • Conservez une copie vierge sans substitution manuelle de symboles.
