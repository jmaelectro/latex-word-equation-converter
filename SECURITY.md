# Security Policy

## Supported versions

The latest branch (`main`) receives security updates.

## Reporting a vulnerability

Please report vulnerabilities privately by email:

- `ecuacionesaword@gmail.com`

Include:
- affected endpoint or feature
- reproduction steps
- potential impact
- suggested mitigation (if available)

## Scope notes
- Do not send sensitive user data in reports.
- Do not run denial-of-service tests against production.
